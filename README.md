# Код
### Шаг 1. Подгрузить библиотеку
```
pip install PyQt5
```
### Шаг 2. Клонирование репозитория и запуск кода
```
mkdir src
cd src
git clone https://github.com/Anette202005/calculator.git .
python calc.py
```
