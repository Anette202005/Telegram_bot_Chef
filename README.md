# Код
### Шаг 1. Подгрузить библиотеку
```
pip install telebot
```
### Шаг 2. Клонирование репозитория и запуск кода
```
mkdir src
cd src
git clone https://github.com/Anette202005/Telegram_bot_Chef.git .
python delicious_food_recipes.py
```
